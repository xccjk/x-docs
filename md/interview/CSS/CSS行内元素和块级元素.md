# CSS行内元素和块级元素

## 元素分类

### 行内元素

- a, b, font, i, img, imput, span, br, label, small, select, textarea
- 行内元素不会独占一行，宽度随元素内容变化
- 行内元素设置宽高无效
- 行内元素设置padding与margin，垂直方向上撑开高度，水平方向生效

### 块级元素

- div, h1-h6, hr, ol, p, address, aside, header, main, footer
- 快级元素独占一行，宽度自动填满父元素宽度
- 块级元素可以设置width和height
- 块级元素可以设置padding和margin

| 行内元素 | 块级元素 |
| -- | -- |
| a, b, font, i, img, imput, span, br, label, small, select, textarea | div, h1-h6, hr, ol, p, address |
