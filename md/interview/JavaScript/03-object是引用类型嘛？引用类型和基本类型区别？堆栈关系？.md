# object是引用类型嘛？引用类型和基本类型区别？堆栈关系？

## object是引用类型嘛

object为引用数据类型

## 引用类型和基本类型区别

### 基本类型

- 基本类型的值是不可变的，复制时是产生一个新的变量
- 基本类型的比较是值的比较
- 基本类型的变量是存放在栈内存中的

### 引用类型

- 引用类型值是可变的，复制时实际复制的是指向堆的引用
- 引用类型的值是保存在堆内的，栈内存放的是值的引用

## 堆与栈的关系

### 堆

- 指堆内存
- 堆中内存是动态分配的，内存大小不一，也不会自动释放
- 引用类型的值是存放在堆中的
- 杂乱无章，方便开辟和存储内存空间

### 栈

- 指栈内存
- 栈是自动分配相对固定的大小的内存空间，并由系统自动释放
- 基本数据类型都是存放在栈上的
- 引用类型的指针也是存放在栈上的
- 线性结构，后进先出
