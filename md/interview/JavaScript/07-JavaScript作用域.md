# JavaScript作用域

## 作用域

### 什么是作用域

- 作用域是用来确定在何处以及如何查找变量/标识符的规则
- 作用域就是查找变量的地方

#### 函数作用域

```javascript
// 函数作用域
function test1() {
    var a = 1
    console.log(a)    // 1
}
test1()
```

函数test1执行时打印了变量a，变量a是在函数内部定义的，这种在函数内部就访问到了变量的称为`函数作用域`

#### 全局作用域

```javascript
// 全局作用域
var b = 1
function test2() {
    console.log(b)    // 1
}
test2()
```

函数test2执行时打印了变量b，在函数内部没有找到变量b时，会到函数外层中查找，找到了就停止查找并且输出。这种通过访问函数外部变量的方式称为`全局作用域`

#### 作用域链

在函数test2的执行过程中，在查找b变量时，先在函数内部查找，没有找到，就到全局作用域中查找，有一个往外查找的过程。好像是顺着一条链条从上往下查找变量，这条链条我们称为作用域链

### 作用域嵌套

```javascript
// 全局作用域
var a = 1
function fn1() {
    // 作用域3
    var b = 2
    function fn2() {
        // 作用域2
        var c = 3
        function fn3() {
            // 作用域1
            var d = 4
            console.log(a)
        }
    }
}
```
